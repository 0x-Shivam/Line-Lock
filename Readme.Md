
# Lock Line

Lock line is web game. it vector puzzel drawing game build for fun. Guide colored dots to their matching destination gates by drawing custom paths, avoiding catastrophic collisions along the way.
Goal is to prevent from getting colid.

## Play Here -- https://linelock.netlify.app/  

# Ai use 
None 
used google for js documentation

# How to play 
 
- *Draw path* Click and drag Mouse and match to it respective color 
- *Guid the dots* Dots will automatically snap onto your drawn lines when they touch them.
- *Score Points* User will get point for sending sucssufully to their end goal
- *Avoid Collision* The most imp thing don;t let dots of differnt color colid with each other 

# Tech 
- **Frontend:** HTML5 Canvas, CSS3
- **Logic:** Vanilla JavaScript (ES6+)
